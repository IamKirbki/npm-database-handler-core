# Interfaces

Core interfaces for creating database adapter packages.

## Interfaces

- **[IDatabaseAdapter](IDatabaseAdapter.ts)** - Database connection and query execution
  - See [documentation](../../../../wiki/adapters/IDatabaseAdapter.md)

- **[IStatementAdapter](IStatementAdapter.ts)** - Statement execution with parameter binding
  - See [documentation](../../../../wiki/adapters/IStatementAdapter.md)

- **[ISchemaBuilder](ISchemaBuilder.ts)** - Schema creation and modification (AbstractSchemaBuilder)
  - See [documentation](../../../../wiki/adapters/ISchemaBuilder.md)
